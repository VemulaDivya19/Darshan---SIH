# 🛕 Darshan – Smart Temple Management Platform

## 📌 About the Project

Darshan is an innovative Digital Temple Management Platform developed for Smart India Hackathon (SIH) 2025.

The project aims to improve the pilgrimage experience by reducing overcrowding, minimizing waiting times, and enhancing safety through Artificial Intelligence (AI), Internet of Things (IoT), and real-time monitoring technologies.

---

## 🚩 Problem Statement

Many popular temples experience:

- Long waiting times
- Heavy crowd congestion
- Safety risks during festivals
- Difficulty in crowd management

Darshan provides a smart digital solution for these challenges.

---

## 💡 Key Features

- QR-based Digital Darshan Pass
- Online Slot Booking
- Live Queue Status
- Crowd Density Monitoring
- AI-based Crowd Prediction
- Emergency Alert System
- Multilingual Support
- CCTV & IoT Integration
